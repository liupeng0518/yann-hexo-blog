TCP/IP-(3)-UDP协议

发表于 2018-06-05 | 更新于 2018-06-12 | 分类于 网络编程 ， TCP/IP | 阅读次数：

UDP是基于IP 报文上层的一个协议，在TCP/IP协议簇中也提及到，其实UDP只是比IP多提供了进程到进程之间的通讯，除此之外并没有提供额外什么功能。

UDP 协议格式

值得注意的点：

  • UDP头部多了源端口号和目标端口号
  • UDP依然受限于IP的报文大小最大65535，还需要减掉20个UDP的报文头，也就是65507
